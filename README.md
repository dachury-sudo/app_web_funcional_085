App Web Funcional - Gestión de Datos

Esta es una aplicación web con frontend y backend, diseñada para gestionar usuarios y productos de manera sencilla y eficiente. Permite agregar, editar y eliminar información, mostrando todo en tablas dinámicas en tiempo real.

Tecnologías

- Frontend:React, JavaScript, HTML, CSS
- Backend:Node.js, Express
- Base de datos:MongoDB

Instalación y Ejecución
Backend
cd backend
npm install
npm start

Frontend
cd frontend
npm install
npm start
